# 💬 Giscussions — Demo Discussions

This repository serves as a **demo backend for the Giscussions fork**, showcasing its features using a single GitHub Discussion.

👉 **Live demo:** https://comments.joyfuldowntime.com

---

## 🌐 About Giscussions

**Giscussions** is a custom fork of giscus—a comments system powered by GitHub Discussions—extended with:

* Improved **discussion UX and layout**
* Enhanced **comment metadata and reactions display**
* Support for **inline edit/delete actions (permission-aware)**
* Detailed **edit history popovers**
* Better **author identity handling (display names + fallbacks)**
* Optional **branding controls**

👉 Original project: https://github.com/giscus/giscus

---

## 🧪 Demo Purpose

This repository is used to:

* Demonstrate **real-world rendering** of the Giscussions UI
* Provide a **live, interactive example** via the demo app
* Test and validate new features in a controlled environment

---

## 💬 How It Works

* A **single discussion thread** is used as the comment source
* The Giscussions client renders this discussion in the demo app
* Visit the live demo to interact with features such as:

  * inline editing and deletion
  * edit history popovers
  * improved metadata layout
  * “View on GitHub” behavior for logged-out users
